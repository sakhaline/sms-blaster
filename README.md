# SMS Blaster
***Automated SMS Outreach Engine*** 


## Usage
```bash
  git clone https://github.com/MrOlegyou/mpig_sms_blaster.git
```
```bash
  cd mpig_sms_blaster
```
```bash
  python3 -m venv .venv
```
```bash
  source .venv/bin/activate
```
```bash
  pip3 install -r requirements.txt
```
```bash
  cd src
```
```bash
  python3 main.py
```
